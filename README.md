# Online Banking System

## Project Overview
A fully functional Online Banking System developed in C programming language as part of the Final Promotion Individual Project for Rockview University.

## Features Implemented

### Core Features:
1. User Registration - Create new bank accounts with validation
2. Fund Deposit - Add money to account with balance update
3. Fund Withdrawal - Withdraw money with sufficient funds check
4. Online Fund Transfer - Transfer between accounts with validation
5. Change Password - Secure password change functionality
6. Display Account Details - Show account info (password hidden)

### Extra Features (Bonus):
- File Handling - All account data is saved to `bank_data.txt`
- Login System - Secure authentication required for banking operations
- Input Validation - Comprehensive error checking and user input validation
- Well-Designed Menu - Clean switch-case based menu system
- Proper Documentation - Code comments and readable structure

## How to Run the Program

### Prerequisites:
- GCC Compiler
- Git bash environment (or any C compiler)

### Compilation:
bash
gcc online_banking_system -o online_banking_system.c
